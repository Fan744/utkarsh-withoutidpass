# Telegram Utkarsh Bot

यह Telegram bot Utkarsh platform से course data extract करता है
और `.txt` files Telegram पर भेजता है।

## Features
- Mobile + Password login
- Batch ID based extraction
- Auto `.txt` file generation
- Telegram delivery

## Requirements
- Python 3.12 (recommended)
- python-telegram-bot 21.6
- requests
- pycryptodome

## Run
```bash
pip install -r requirements.txt
python bot.py
